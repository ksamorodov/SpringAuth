# SIMPLE AUTH

### Локальный запуск

#### services
`docker-compose up` - поднимет базу данных и nginx

#### backend
`mvn spring-boot:run`

#### frontend
1. `cd frontend`
2. `npm install`
3. `npm install -g @angular/cli`
4. `ng serve`

#### Приложение
После запуска приложение будет доступно в браузере по адресу `http://localhost`
